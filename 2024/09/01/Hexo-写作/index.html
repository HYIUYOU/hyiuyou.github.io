Hexo 写作命令

  1. cd到写作目录

  2. 新建一个新的文章

    这会在 source/_posts 目录下生成一个新的 Markdown 文件，你可以编辑这个文件以撰写你的文章。

  3. 发布文章

    1. 首先生成静态文件，类似CPP中的编译

    2. 本地预览（可选）

    3. 部署blog

  4. 修改/删除文章

    1. 找到要修改/删除的文章

      Hexo 的博客文章存储在 source/_posts 目录下。每篇文章对应一个 Markdown 文件，文件名通常与文章标题相对应。

    2. 修改/删除文章文件

      进入 source/_posts 目录，找到要修改/删除的博客文章文件，然后修改/删除该文件。

    3. 清除生成的文件并重新生成静态文件

    4. 部署更新
